///UI 구성///

로봇 시스템 로그(작업 과정, 로봇 상태)

로봇암 좌표, 선속도, 각속도, 베이스, 엔드이펙터

긴급정지, 비상복구 버튼


///흐름///

AP가 ROS2로 웹 DB에 로그 보냄

웹 DB에 데이터 저장

화면. 로봇 작업여부(온, 오프, 충돌, 시스템 오류)

화면. 로봇 상태(작업 프로세스)

화면. 로봇 상태(가동시간)

화면. 로봇 위치(좌표)

화면. 시스템명, 시스템 버전명, 기기명

화면. DB 버튼

화면. 웹 DB에 화면출력

화면. 웹 DB에 로봇암 현재모양 출력

버튼. 로그출력

버튼. 직전 60초간 작업내용 10Hz 단위로 출력

웹에 접속하면 UI가 출력

데이터 받아야 함

데이터를 디비에 저장

디비에서 데이터 출력
